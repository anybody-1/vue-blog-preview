
## 项目名称：vue.js实现多人共享博客

## 项目介绍

本项目做一款多人共享博客，包含首页、用户文章列表、登录、注册、个人管理、编辑、发布等功能。

## 技术栈
 
Vue.js 技术栈：vue-cli/vue2/axios/vue-router/vuex/es6/npm

## 项目收获

在本项目中我学会了运用vue-cli创建项目模板，利用vuex进行vue.js应用进行状态管理。 利用axios进行资源请求，以及利用vue-router进行路由管理

## 问题及难点

1. vue 组件之间的相互引用
2. 路由切换
3. vuex的设置及调用
4. ES6语法在webpack中的应用
5. computed-计算属性的设置
